Glossary

!DOCTYPE html
A tag !DOCTYPE html define a versão do HTML que está sendo usada na página web, permitindo ao navegador interpretar corretamente o código HTML.
html
A tag define o início e o fim do documento HTML, envolvendo todo o conteúdo da página web.
head
A tag contém informações sobre o documento HTML, como metadados, scripts e links para estilos, mas não é exibida na página web.
body
A tag contém o conteúdo visível da página web, incluindo texto, imagens, links e outros elementos que são exibidos no navegador.
